Ce projet est développé dans le cadre de la formation du système d'exploitation en troisième année du département informatique. Le but du projet est de pratiquer les connaissances que nous avons acquises durant les séances du système d'exploitation. 


Pour cela, nous sommes demandés à développer une application sous Android pour contrôler un petit robot au moyen d'un microcontrôleur nommé AndroPOD. Donc il nous faut étudier à la fois :
\bigskip
\begin{itemize}
	\item Le développement sous le système Android
 	\item L'interface de programmation de la carte AndroPOD
 	\item Le format des commandes pour contrôler le petit robot
\end{itemize}
\bigskip


Notre travail, ainsi que ce rapport, est organisé selon ces trois parties.
